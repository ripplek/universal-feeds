# Security Policy

## Supported Versions

The `main` branch is supported.

## Reporting a Vulnerability

Please **do not** file public issues for security-sensitive reports.

Instead:
- Email: security@ripplek.com
- Include: affected version/commit, reproduction steps, and impact assessment.

## Notes

This project may interact with third-party platforms (RSS/HTML/public pages and optional local cookie-based auth via `bird`).

- Never commit cookies/tokens.
- Treat any auth integration as user-controlled and local-only.
